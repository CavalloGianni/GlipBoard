# GlipBoard 0.1.0

Prima release pubblica di GlipBoard.

## Incluso nella release

- finestra principale GTK4/libadwaita
- cronologia clipboard testuale
- tray di supporto per accesso rapido
- impostazioni base per numero massimo elementi e comportamento iniziale
- installazione locale per utente Pop!_OS

## Target

- Pop!_OS
- Wayland

## Note

- release iniziale orientata a un uso reale ma ancora in evoluzione
- packaging `.deb` e materiali visuali della repository arriveranno in uno step successivo
