# GlipBoard v0.1.0

Prima release pubblica di GlipBoard.

## Cosa include

- applicazione desktop GTK4/libadwaita
- cronologia locale dei testi copiati
- tray helper con accesso rapido a finestra e ultimi elementi
- preferenze base per limite elementi, avvio iniziale e autostart
- supporto Wayland tramite `wl-clipboard`
- supporto X11 tramite `xclip`
- selezione automatica backend clipboard disponibile
- pacchetto `.deb` per installazione desktop

## Privacy

- cronologia appunti salvata solo in locale sul PC dell'utente
- storage spostato sul path standard `~/.local/share/glipboard/`
- permessi locali piu restrittivi per ridurre esposizione ad altri utenti/processi locali
- rimossa stampa nel terminale del contenuto copiato

## Note

- release pubblica iniziale consigliata per GlipBoard
- target principale: Pop!_OS e desktop Linux compatibili con GTK
- GlipBoard non va considerato archivio sicuro per password, token o altri segreti
