# GlipBoard 0.1.1

Compatibility update for Ubuntu and X11-based sessions.

## Included in this release

- clipboard backend support for X11 via `xclip`
- automatic backend selection between Wayland and X11
- updated Debian package dependencies for broader Linux desktop compatibility

## Target

- Pop!_OS
- Ubuntu
- Wayland
- X11

## Notes

- this release is focused on clipboard compatibility improvements
- the `_apt` warning shown during local `.deb` installation is not the root cause of the clipboard issue
- Wayland support remains the primary path already validated on Pop!_OS
